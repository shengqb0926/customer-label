## 🔍 核心功能

### 1. 智能标签推荐
- **规则引擎**: 基于预定义业务规则推荐标签
- **聚类引擎**: 使用 K-Means 算法发现客户群体特征
- **关联引擎**: 基于 Apriori 算法挖掘标签关联关系
- **融合引擎**: 多引擎结果加权融合

### 2. 标签质量评分
- **覆盖率**: 标签适用的客户比例
- **区分度**: 标签对客户分群的区分能力
- **稳定性**: 标签值随时间的稳定性
- **业务价值**: 标签对业务决策的指导意义

### 3. 冲突检测
- **命名冲突**: 检测同名或高度相似的标签
- **逻辑冲突**: 识别互斥的标签组合
- **冗余检测**: 发现含义重复的标签

### 4. 用户反馈
- **反馈收集**: 记录用户对推荐的采纳/拒绝
- **效果分析**: 分析推荐质量和采纳率
- **持续优化**: 基于反馈优化推荐算法

## 🏗️ 技术架构

### 技术栈
```yaml
后端:
  runtime: Node.js 18.x
  language: TypeScript 5.0+
  framework: NestJS 10.x
  orm: TypeORM 0.3.x
  
机器学习:
  - TensorFlow.js (聚类算法)
  - Simple-statistics (统计分析)
  
数据存储:
  database: PostgreSQL 15
  cache: Redis 7.x
  queue: Bull
  
前端:
  framework: React 18.x
  ui: Ant Design 5.x
  charts: ECharts
```

### 系统架构
```
客户端层 → API 网关 → 应用服务层 → 业务逻辑层 → 数据访问层 → 数据存储层
```

### 核心模块
- `recommendation/` - 推荐引擎（规则、聚类、关联）
- `scoring/` - 评分引擎（四个维度评分器）
- `conflict/` - 冲突检测器（命名、逻辑、冗余）
- `feedback/` - 反馈管理模块
