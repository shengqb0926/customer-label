## 📝 六、附录：故事编写指南

### 好的用户故事特征 (INVEST)

- **I**ndependent（独立的）：故事之间无依赖
- **N**egotiable（可协商的）：细节可讨论
- **V**aluable（有价值的）：对用户有明确价值
- **E**stimable（可估算的）：能评估工作量
- **S**mall（小的）：可在一个迭代内完成
- **T**estable（可测试的）：有明确验收标准

### GWT 格式模板

```gherkin
Feature: 功能名称
  As a <角色>
  I want <功能>
  So that <价值>

  Scenario: 场景名称
    Given <前置条件>
    When <触发事件>
    Then <预期结果>
    And <额外结果>
```
